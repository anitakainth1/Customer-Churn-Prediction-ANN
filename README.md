# Customer Churn Prediction Using ANN

## Overview
This project predicts customer churn using Artificial Neural Networks (ANN) built with TensorFlow and Keras.

## Technologies Used
- Python
- TensorFlow
- Keras
- Scikit-learn
- Pandas
- NumPy
- Matplotlib

## Features
- Data preprocessing
- Feature scaling
- ANN model building
- Model training and validation
- Overfitting analysis
- Prediction visualization

## Model Performance
The model was trained using binary crossentropy loss and Adam optimizer.

## Future Improvements
- Hyperparameter tuning
- Streamlit deployment
- ROC-AUC analysis
- Model optimization
